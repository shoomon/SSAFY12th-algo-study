## Git 기본 명령어

### 로컬 폴더 - 깃 연결
git으로 관리할 가장 상위 폴더 위치에 연결
`git init`

저장소 연결
`git remote add origine 저장소 url `

### 업로드 시
workspace 내 모든 파일을 staging 영역으로 업로드
`git add .`

repository에 업로드 전 임시 저장
`git commit -m "message"`

repository에 업로드
git push origine main
